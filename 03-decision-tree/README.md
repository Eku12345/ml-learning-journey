# Decision Tree Classifier - Titanic Survival Prediction

## Objective

Build a Decision Tree classifier to predict passenger survival using the Titanic dataset and understand how tree-based models make decisions.

---

## Skills Demonstrated

* Exploratory Data Analysis (EDA)
* Missing Value Handling
* Feature Selection
* Encoding Categorical Variables
* Decision Tree Classification
* Tree Visualization
* Feature Importance
* Hyperparameter Tuning
* Cross Validation
* Model Evaluation

---

## Dataset

Titanic Dataset (Seaborn)

---

## Algorithm

* Decision Tree Classifier

---

## Evaluation Metrics

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
* ROC-AUC

---

## Key Learnings

* How Decision Trees split data
* Gini Index vs Entropy
* Information Gain
* Overfitting and Underfitting
* Tree Pruning
* Feature Importance
* Why Decision Trees do not require feature scaling
